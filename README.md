## 背景

视觉组的相关知识比较深入，难度也不小。因此，为视觉组感兴趣的同学设立了Vision_Group_Guideline，展示招新任务和学习路线等相关内容。

我们通过开发任务确定学习路线，通过学习路线制定招新任务。希望大家能学有所获。
